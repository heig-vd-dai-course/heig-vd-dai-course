## Feedback

This directory contains all previous feedback provided by students through
[Framaforms](https://framaforms.org/) and [GAPS](https://gaps.heig-vd.ch/).

These are used to gather feedback from people who have taken the course and to
improve the quality of the teaching unit.

The results are published so that interested parties can consult the feedback
and evaluations and see the improvements that have been made as a result.
