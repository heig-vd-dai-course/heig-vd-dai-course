## Objectives

- Review the main concepts seen during the semester.
- Prepare for the final exam.
- Answer students' questions.
- Provide guidance for further learning.

## Teaching and learning methods

The teaching and learning methods used to facilitate the course are as follows:

- Lecture.
- Group discussions.

## Evaluation methods

The evaluation is based on the following criteria:

- Ability to respond accurately.
- Ability to argue.
- Ability to synthesize.

Feedback is provided as follows:

- Corrected evaluation with comments.
